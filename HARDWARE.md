# Hardware – GPIO Map

> **Headless-конфигурация.** OLED и энкодер удалены, их пины переназначены на мотор
> и концевики. Управление и статус — через веб-интерфейс (точка доступа WiFi `dozator`).

## 2.1 Текущая привязка (активна)

| Функция      | Wemos | GPIO  | Режим / подключение                       |
|--------------|-------|-------|-------------------------------------------|
| Stepper STEP | D2    | GPIO4 | Output → A4988 STEP                        |
| Stepper DIR  | D1    | GPIO5 | Output → A4988 DIR                         |
| Концевик TOP | D5    | GPIO14| `INPUT_PULLUP`, на **GND** (active-LOW)    |
| Концевик A   | RX    | GPIO3 | `INPUT_PULLUP`, на **GND** (теряется Serial RX) |
| Концевик B   | D7    | GPIO13| `INPUT_PULLUP`, на **GND** (active-LOW)    |
| Концевик BOT | D6    | GPIO12| `INPUT_PULLUP`, на **GND** (active-LOW)    |
| Кнопка СТОП  | D3    | GPIO0 | `INPUT_PULLUP`, на **GND** (подтяжка 10к) ⚠️ boot |

**Все концевики и кнопка — единообразно на GND, срабатывание = LOW.**
Микрошаг A4988 (MS1/2/3) — **зашит перемычками на +5V** (1/16), МК не управляет.
EN драйвера — на **GND** (всегда включён). RESET+SLEEP — на **+5V**.

Свободны: **D0/GPIO16, D4/GPIO2, D8/GPIO15, TX, A0**.

### ⚠️ Единственный нюанс — кнопка на boot-пине
Кнопка СТОП на D3/GPIO0 (boot-пин): при **включении** платы кнопка должна быть **разомкнута**
(в покое так и есть). Подтяжка 10к держит GPIO0 в HIGH при старте — норма. Концевики теперь
на не-boot пинах, так что «шприц прижал концевик при старте» больше не мешает загрузке.
TOP на RX (GPIO3): приём Serial не работает (лог по TX — работает).

## 2.2 Схема подключения драйвера A4988

```
Wemos D1 mini        A4988
   D2 (GPIO4)  ────── STEP
   D1 (GPIO5)  ────── DIR
   GND         ────── EN       ← ВКЛЮЧАЕТ драйвер (НЕ на +5V!)
   +5V ──┬─────────── RESET
         └─────────── SLEEP    (RST и SLP соединены, на +5V)
   +5V ──┬─────────── MS1
         ├─────────── MS2      (все три на +5V = 1/16)
         └─────────── MS3
   +5V         ────── VDD
   +12V        ────── VMOT     (+ конденсатор 100мкФ по VMOT/GND)
   GND         ────── GND
   1A/1B/2A/2B ────── обмотки мотора (пары; в паре ~2–5 Ом)
```
Vref подстроечником — под ток мотора (~0.3–0.5 В для начала).

## 2.3 Концевики и кнопка (все на GND)

```
   TOP  → D5 (GPIO14) ─┐
   A    → RX (GPIO3)  ─┤  вторые контакты → общий GND
   B    → D7 (GPIO13) ─┤  срабатывание = замыкание (LOW)
   BOT  → D6 (GPIO12) ─┤
   СТОП → D3 (GPIO0)  ─┘
```

Концевики нормально-разомкнутые (NO). Для NC — инвертировать логику в `RealSwitches::tick()`.

## 2.4 История: снятая периферия

OLED (SSD1306, I2C на D1/D2) и квадратурный энкодер (D5/D6/D7) были в ранней версии,
затем удалены — устройство headless, управление через веб. Их пины переназначены на
мотор и концевики (таблица §2.1).

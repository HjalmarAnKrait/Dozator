# Локальный мок UI (без микроконтроллера)

Запускает веб-интерфейс из `../data` на компьютере и имитирует прошивку
(WebSocket-протокол + автомат состояний), чтобы тестировать и смотреть UI
без платы.

## Запуск

```bash
cd dev
npm install      # один раз (ставит ws)
npm start        # node mock-server.js
```

Открыть **http://localhost:8000** в браузере.

## Что имитируется

- Полный цикл: PARKING → PARKED → CHARGING → CHARGED → DOSING → DONE.
- Концевики срабатывают по таймерам (как `SimulatedSwitches` в прошивке);
  живые `rawSwitches` обновляются → отладочная панель TOP/A/B/BOT мигает по ходу.
- Команды (`Старт зарядки`, `ПУСК`, `Стоп`, `Новый цикл`), поля ввода
  (`direct_set`) и редактирование пресетов (`set_preset`) работают.
- Дозирование ускорено в `TIME_FACTOR` раз (по умолчанию 30×), чтобы не ждать
  реальные минуты. Параметр — вверху `mock-server.js`.

## Важно

- Папка `dev/` НЕ входит в прошивку и НЕ заливается на устройство (`uploadfs`
  берёт только `data/`). Это чисто инструмент разработки.
- Протокол синхронизирован с `WEBSOCKET_PROTOCOL.md`. Если меняешь протокол в
  прошивке — поправь и `mock-server.js`.
